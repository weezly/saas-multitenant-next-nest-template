# @saas/shared

Gemeinsame Types und Utilities für das SaaS Monorepo.

## Installation

```bash
pnpm install @saas/shared
```

## Verwendung

```typescript
import { User, Tenant, ApiResponse } from '@saas/shared';
import { generateSlug, isValidEmail } from '@saas/shared/utils';
```

## Struktur

- `types.ts` - Gemeinsame Type Definitionen
- `utils.ts` - Utility Funktionen
